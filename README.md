Análisis de Deserción Escolar: 
Este proyecto analiza los factores asociados a la deserción escolar utilizando herramientas de análisis de datos en Python.

Objetivo:
Identificar patrones y variables que influyen en el abandono y la permanencia de estudiantes.

Herramientas utilizadas:
Python (pandas, matplotlib, seaborn).
Google Colab.
Power BI.

Principales hallazgos:
La necesidad de trabajar es uno de los principales motivos de deserción.
La asistencia y el rendimiento académico influyen en la permanencia.
La mayoría de los estudiantes se encuentra en edades donde combinan estudio y trabajo.

Conclusión:
El análisis permite identificar factores clave que pueden utilizarse para diseñar estrategias de retención escolar y sistemas de alerta temprana.

Visualización:
Se desarrolló tres dashboard en Power BI para visualizar los principales indicadores de deserción y permanencia.
